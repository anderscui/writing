Loading and preprocessing the data

First, I use the following codes to load data

data <- read.csv("activity.csv")
# convert the data type of date variable to Date.

What is mean total number of steps taken per day?

For making the histogram of the total numbers of steps taken each day, I need to calculate the sum of steps by aggregate function.

date_steps_ag <- aggregate(steps ~ date, data = data, 
                           sum, na.rm = TRUE,
                           na.action = NULL)
hist(date_steps_ag$steps,
     xlab = "steps",
     ylab = "Frequency",
     main = "Total number of steps per day")


# calculate the mean and median
date_steps_mean <- round(mean(date_steps_ag$steps))
date_steps_median <- round(median(date_steps_ag$steps))

Now I know the mean and median of total number of steps taken per day are 9354 and 1.0395 × 104, respectively.

plot(interval_steps_ag$interval, interval_steps_ag$steps,

Imputing missing values

incomplete <- data[!complete.cases(data),]

There are 2304 rows with NA values.

Next, I will store the filled dataset in new_data variable, and fill the missing steps with the mean of each 5-minute interval.

# copy data to new dataset
new_data <- data

# add new column for sorting
new_data$row_id <- 1:nrow(new_data)
# merge and sort
new_data <- merge(new_data, interval_steps_ag,
                by.x = "interval", by.y = "interval",
                all = T)
new_data <- new_data[order(new_data$row_id),]

# assign the missing steps with the mean of the corresponding interval
na_rows <- which(is.na(new_data$steps.x))
new_data$steps.x[na_rows] <- new_data$steps.y[na_rows]

# clear up the data
new_data <- new_data[, 1:3]
names(new_data)[2] <- "steps"

Are there differences in activity patterns between weekdays and weekends?

In this part, I will show some activity patterns based on the new dataset. I use the codes below to add a new factor to indicate weekday and weekend.

weekdays <- weekdays(new_data$date, abbreviate = TRUE)
new_data$weekdays <- ifelse(weekdays %in% c("Sun", "Sat"), 
                        "weekend", "weekday")
new_data$weekdays <- as.factor(new_data$weekdays)

Then we can get the summary information of activity patterns in weekdays and weekends with a simple plot, which is plotted by the lattice system.

library(lattice)
# get aggregation results of steps by interval and weekdays.
inter_day_steps_ag <- aggregate(steps ~ interval + weekdays, 
                               data = new_data, 
                               mean)

# show the relation of steps and interval by weekdays.
xyplot(steps ~ interval | weekdays, 
       data = inter_day_steps_ag, 
       layout = c(1, 2),
       type = "l",
       xlab = "interval",
       ylab = "Number of steps")

Through this plot, we can get several interesting results. The highest point occures at 8:00 AM - 9:00 AM both in weekdays and weekends. But after that time, the average steps of weekends are obviously higher than weekdays, which may imply that this person likes hanging out in weekends, but he/she has to stay in the office in weekdays.
